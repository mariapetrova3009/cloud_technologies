# 4. Описание сервисов AWS, встречающихся в биллинге

## 4.1. Amazon S3 — объектное хранилище

Amazon S3 — масштабируемое объектное хранилище для хранения файлов любого размера: медиа, логов, дата-сетов для аналитики.  
Используется для хранения данных, обеспечения их доступности и долговечности.

**Российские аналоги:**  
Yandex Object Storage, VK Cloud Storage, S3-совместимые хранилища Selectel.


## 4.2. Amazon MSK — Managed Streaming for Apache Kafka

Amazon MSK — управляемый сервис Kafka, который берет на себя масштабирование, отказоустойчивость и обслуживание брокеров.  

Используется для обмена сообщениями между микросервисами, потоковой обработки данных и логирования.

**Российские аналоги:**  
Yandex Managed Kafka, VK Cloud Kafka.

## 4.3. Amazon Personalize — сервис рекомендаций

Amazon Personalize — облачный сервис машинного обучения, который автоматически строит персональные рекомендации 
Используется для построения рекомендаций товаров, фильмов, контента.

**Российские аналоги:**  
VK Cloud ML, SberCloud ML Space.

## 4.4. Amazon Polly — сервис синтеза речи

Amazon Polly преобразует текст в речь с помощью моделей нейронной озвучки.  
Используется для генерации аудио, озвучки сервисов, IVR-систем, доступности для людей с ограничениями.

**Российские аналоги:**  
Yandex SpeechKit, VK Cloud Speech.

## 4.5. Amazon ES (OpenSearch Service)

Amazon OpenSearch Service — управляемая поисковая система на основе Elasticsearch/OpenSearch.  
Применяется для анализа логов, полнотекстового поиска, мониторинга.

**Российские аналоги:**  
VK Cloud Big Data, Yandex DataLens (частично), Selectel Elasticsearch-кластер.
